Растровые спрайты

Спрайт это технология объединения множества изображений в один файл и извлечение из общего файла требуемого изображения для размещения на web-странице.

Необходимость использования этой технологии вызвана ограничениями браузера на общее количество одновременных запросов к серверу (не более 17) и на ограничение запросов к одному серверу (зависит от браузера).

Это ограничение приводит к тому, что скорость первой загрузки страницы зависит не только от размера файла, но и от количества файлов. И одним из самых эффективных способов добиться загрузки страницы в пределах 2,5 секунд (среднее время, после которого пользователь переходит на новую страницу) является уменьшение числа файлов. CSS или JS файлы подвергаются простой склейке. Изображение использует необычный прием, основанный на свойстве background - рассмотрим его подробнее.

  1. Разместить изображения в одном файле с обязательной дистанцией между каждым новым изображением (обычно используют вертикальное или горизонтальное расположение, но самым оптимальным является диагональное расположение)
  2. Создать элементы или псевдоэлементы с указанием точного размера, совпадающим с размером нужного изображения из файла
  3. Для каждого из элементов задать путь к общему файлу с изображениями background: url(...) no-repeat;
  4. С помощью свойства background-position задать смещение по горизонтали и вертикали так, чтобы фоном оказался участок с нужным изображением из всех доступных в файле.

Для создания эффектов при hover можно добавить в спрайт еще ряд изображений и показывать их при наведении. Однако это увеличивает размер файла и поэтому стоит подобрать эффекты CSS3 для изображений, к примеру filter.

Автоматизация создания спрайтов

Существует много способов, позволяющих автоматизировать довольно рутинную работу по созданию спрайтов. Например, онлайн-сервис sprite-generator создаст готовый спрайт в формтате png на основе загруженных вами изображений. В программе JavaScript будет рассказано и о других способах автоматизации создания спрайтов.
