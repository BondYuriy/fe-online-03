Полезные команды консоли:

  • cd - переход в заданную папку
  • ls - показывает список файлов текущей папки
  • mkdir - создать папку
  • rmdir - удаление пустой папки
  • touch - создать файл
  • mv - перемещение файлов/папок
  • rm - удаление файлов/папок

Команды git:

  • git init - инициализирует локальный репозиторий в текущей папке
  • git clone - клонирует удаленный репозиторий в текущую папку
  • git status - отображает состояние рабочего каталога и stage
  • git add - добавляет новые или измененные файлы в stage
  • git reset - удаляет файл из под git-контроля
  • git commit - делает коммит подготовленных файлов в историю проекта
  • git push - отправляет историю проекта (коммиты) на удаленный репозиторий
  • git log - отображает историю коммитов локального репозитория
  • git remote -v - отображает с каким удаленным репозиторием связан локальный
